> v1.1 | 2026-07-10 补 DS 低俗色情 + ZH 引战谩骂两类（抖音第一红线原缺）+ 引流变形词变体

## 变形词归一化规则（扫描前置）

> 抖音规避手段常用同音字/谐音字/符号插入/全角半角/繁简变形绕过关键词检测。
> 扫描每类前，先把文案做变形归一化再匹配，避免"加薇信""加ＶＸ""加★微★信"等全部漏检。

| 变形类型 | 归一化方式 | 示例 |
|---------|-----------|------|
| 谐音/同音字替换 | 常见引流词谐音变体列入 YL 词库直接匹配 | "薇信""徵信""喂心"→微信 |
| 全角半角混用 | 全角字母数字（ＶＸ）转半角再匹配 | "加ＶＸ"→加VX |
| 符号插入干扰 | 容忍词内插入 0-3 个非汉字符号 | "加★微★信"匹配"加微信" |
| 繁简混用 | 繁体转简体（"聯係"→联系）再匹配 | 视情况，口播少用 |

> 注：拼音首字母/拆字等更激进变形（如"WX""微x"）暂不检测——口播作者文案场景罕见，
> 过度检测会误伤正常词，按宁漏勿误原则留待实际限流案例触发再补。

## DS 低俗色情（高风险）

> 抖音第一红线。擦边描述、性暗示、软色情即限流，硬色情直接封号。
> AI/tech 口播较少触达，但"擦边玩梗""低俗比喻"要警惕。结合上下文：科普语境的生理词不算，引流/挑逗语境才算。

### 性暗示/软色情

| 关键词 | 说明 | 建议替换 |
|--------|------|---------|
| 约炮/一夜情 | 明确低俗 | 删除 |
| 搞涩涩/搞黄色 | 低俗谐音玩梗 | 删除 |
| 涩涩/网袜 | 性暗示谐音/借指 | 删除 |
| 老司机发车 | 低俗隐喻 | 删除或改为"经验分享" |
| 开车（性暗示语境） | 软色情隐喻 | 删除或改语境 |
| 福利（色情语境） | 软色情借指 | 删除 |
| 懂的都懂（性暗示） | 低俗暗示 | 删除 |

### 擦边/低俗比喻

## ZH 引战谩骂（高风险）

> 平台红线：人身攻击、地域攻击、拉踩引战、脏话谩骂触发限流甚至禁言。
> AI/tech 口播易在"对比""测评"场景踩拉踩线，务必中立陈述。

### 人身攻击/脏话

| 关键词 | 说明 | 建议替换 |
|--------|------|---------|
| 傻逼/智障及谐音（伞兵/shabi） | 脏话谩骂 | 删除 |
| 脑残/废柴/垃圾（骂人语境） | 人身攻击 | 就事论事不骂人 |
| 滚/闭嘴（攻击语境） | 攻击性语言 | 改为客观陈述 |
| 狗屎/扯淡（骂产品） | 谩骂 | "有待改进"/"存在问题" |

### 地域/群体攻击

| 关键词 | 说明 | 建议替换 |
|--------|------|---------|
| 地域黑（"某地人都是X"） | 地域歧视红线 | 删除 |
| 性别攻击（"男人/女人都X"） | 性别对立引战 | 删除 |
| 学历歧视（"小学毕业懂什么"） | 群体攻击 | 删除 |

### 拉踩引战（AI 测评高频）

| 关键词 | 说明 | 建议替换 |
|--------|------|---------|
| 吊打/完爆/秒杀（贬竞品） | 拉踩引战 | "在X方面表现更好" |
| XXX已死/凉了 | 引战唱衰 | 中立分析现状 |
| 不用XXX是傻子 | 道德绑架引战 | "可以根据需求选择" |
| 只有傻子才用XXX | 人身攻击+引战 | 删除 |
| 洗地/水军（无证据指控） | 引战指控 | 删除或给证据 |
