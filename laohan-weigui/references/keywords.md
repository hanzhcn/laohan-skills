# 抖音违规词库

> v1.0 | 初始版本，基于抖音社区规则 + 常见限流案例整理
> 维护方式：遇到实际限流/禁言时，把具体词加入对应分类

## YL 引流违规（高风险）

### 直接引流词

| 关键词 | 说明 | 建议替换 |
|--------|------|---------|
| 加我微信 | 明确站外引流 | 删除 |
| 加V | 引流微信 | 删除 |
| 加VX | 引流微信 | 删除 |
| 扫码 | 引导扫码加联系方式 | 删除或改为"关注我" |
| 私信我 | 可能触发引流判定 | "评论区留言"或"关注我后续更新" |
| 私聊我 | 同上 | 同上 |
| 我的主页有 | 隐性引流 | 删除 |
| 主页链接 | 引流 | 删除 |
| 主页简介 | 引导看主页联系方式 | 删除 |
| 评论区置顶 | 引导看评论区信息 | 删除 |
| 进群 | 引流到社群 | "持续关注" |
| 入群 | 同上 | 同上 |
| 群号/QQ群 | 直接引流 | 删除 |
| 淘宝搜 | 引流电商 | 删除 |
| 拼多多搜 | 引流电商 | 删除 |
| 买的话 | 引导购买 | 删除 |
| 多少钱/什么价格 | 可能触发电商审核 | 根据语境调整 |

### 联系方式模式

| 模式 | 正则 | 说明 |
|------|------|------|
| 微信号 | `微信[：:]?\s*[a-zA-Z0-9_-]{6,20}` | 直接出现微信号 |
| 手机号 | `1[3-9]\d{9}` | 直接出现手机号 |
| QQ号 | `QQ[：:]?\s*\d{5,12}` | 直接出现QQ号 |
| 链接 | `https?://` | 任何URL（除抖音站内） |

## MG 敏感话题（高风险）

> 注意：这类词需要结合上下文判断，以下列出常见触发词，但并非出现就违规

### 政治敏感

| 关键词 | 说明 |
|--------|------|

### 社会敏感

| 关键词 | 说明 | 建议替换 |
|--------|------|---------|

| 歧视性用语 | 平台红线 | 删除 |

## PT 平台特有限制（中风险）

### 隐性限流规则

| 规则 | 说明 | 检测方式 |
|------|------|---------|
| 重复内容 | 同一文案发多次 | 提醒不要重复发布 |
| 频繁@他人 | 互动异常 | 检查@数量 |
| 过多话题标签 | #超过5个可能降权 | 计数#标签 |
| 过长文案 | 超出推荐字数 | 口播稿建议350-1200字 |
| 同质化内容 | 与之前视频高度相似 | 提醒差异化 |
| 频繁删除重发 | 被判为刷量 | 提醒不要删了重发 |

### 特殊时期限制

| 时期 | 限制 | 说明 |
|------|------|------|
| 重大会议期间 | 内容审核更严 | 提前检查 |
| 节假日前 | 营销内容受限 | 注意措辞 |
| 热点事件中 | 相关内容审核加严 | 谨慎发言 |
