# 秋芝2046 封面实测分析

> 2026-04-27 | 通过B站API抓取5张封面 + AI视觉分析验证

## 分析方法

1. B站 API `/x/web-interface/view?bvid=XXX` 获取封面图 URL（无需 WBI 签名）
2. `analyze_image` 逐张分析人物比例、构图、视觉元素
3. 量化头部占画面高度百分比

## 核心发现：头部比例

| 指标 | 实测值 | 之前假设 |
|------|--------|---------|
| 头部占画面高度 | **16-33%**（中近景约25-30%） | 40% |
| 最常见范围 | **25-30%** | — |
| 极端大头 | 不使用 | — |

**结论**：秋芝不是极端大头照，是中近景构图。头部突出靠的是：
1. **中近景构图**（head + upper body 占画面主体）
2. **黄色光晕**（头部周围常有暖黄色 glow halo）
3. **夸张表情**（张大嘴、瞪眼、侧头等戏剧性表情）
4. **中心位置**（人物始终居中，文字左右对称）

## 秋芝视觉设计要素（已验证）

### 固定不变的部分
- **背景**：纯白 #FFFFFF 或极浅灰 #F5F5F5
- **文字**：亮黄 #FFD700 + 粗黑描边，左右对称，卡通风格粗体
- **人物**：真人照片（非卡通/动漫），穿角色扮演服装+夸张道具
- **排版**：人物居中，文字左右对称框住头部

### 每期变化的部分
- **角色身份**：侦探/科学家/厨师/宇航员/矿工等（29种见 styles.md）
- **道具**：与主题相关的夸张道具（放大镜、试管、冲浪板等）
- **表情姿势**：每期不同的互动姿势和戏剧性表情
- **场景细节**：主题相关的小元素（蒸汽、电路、光效等）

### 为什么秋芝封面有效
1. **白底高对比**：在B站/抖音信息流中极度醒目
2. **对称大字**：标题拆成两半左右对称，一眼识别
3. **人物+道具**：真人照片（不是卡通）增加可信度，cosplay增加趣味
4. **表情夸张**：情绪传染，吸引点击
5. **系列一致性**：固定模板降低认知成本，观众一看就知道是秋芝

## B站 vs 抖音封面对比（2026-04-27 实测）

同一内容在两个平台使用**完全不同**的封面策略：

| 元素 | B站封面 | 抖音封面 |
|------|--------|---------|
| 背景 | 纯白 #FFFFFF / 极浅灰 | 彩色主题背景（木质货架/深空/蓝天等） |
| 文字位置 | 标题拆半，左右对称框住头部 | 顶部居中，短标签（"AI大全"/"AI剪辑"/"Agent天堂"） |
| 文字内容 | 完整标题拆成两半 | 简短分类标签（2-4字） |
| 头部比例 | 25-30% | 15-30%（部分更大） |
| 文字颜色 | 亮黄 #FFD700 + 粗黑描边 | 相同（一致） |
| 服装/道具 | cosplay + 夸张道具 | 相同（一致） |
| 表情 | 夸张戏剧性 | 相同（一致） |

**结论**：秋芝在两个平台用不同策略，但共享核心元素（黄色粗字+cosplay+夸张表情）。

**对我们的提示词影响**：当前 SKILL.md 模板基于B站白底+对称文字风格，适用于B站/小红书横版。抖音竖版可考虑用彩色主题背景+顶部居中标签的变体，但暂时不做，等有需求再说。
