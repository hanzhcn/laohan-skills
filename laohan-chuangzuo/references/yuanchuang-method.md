# 热点转译选题法

从实时热点中提炼老韩原创选题的方法论。

核心思路：**热点 = 选题来源（100%），老韩视角 + style.md = 差异化。** "原创"不是从零生造，是在别人都说同一个热点的时候，你说得不一样。

**转译公式：你的真实 AI 经历 + 热点话题 = 你专属的选题。** 观众要的不是 AI 知识，是 **AI 跟我的关系**。

**铁律：标准迁就爆款，不迁就创作者。** 观众不关心你有没有经历过，他们关心的是你说的话像不像真话。

## 转译案例

别人在"报道热点"，老韩在"翻译成我的经历"：

| 热点 | 别人怎么讲 | 老韩怎么转译 |
|------|-----------|------------|
| 35岁男子被AI取代通知（知乎242万热度）| 报道新闻、分析法律赔偿 | "我就是那个用AI取代别人的人"——8个agent帮我做内容，一个顶几个人。讲我用AI做掉了什么岗位 |
| AI大模型中文比英文更费Token（36Kr）| 技术分析 | "我月烧50亿token，告诉你中文到底贵在哪"——老兵视角算账：到底值不值 |
| CTO转身去Anthropic当工程师（36Kr）| 行业新闻 | "老韩在互联网十几年见过几次周期"——不是技术分析，是职业选择分析 |
| 假期出游被AI坑惨了（36Kr）| 罗列翻车案例 | "AI帮我规划过旅行，坑了我多少钱"——真金白银的账 |

**模式：** 转译的关键不是换个说法讲同一件事，是**换成你的身份去讲这件事**。

## 三个必答题

对每个候选热点必须回答，三个都通过才成立：

1. **跟 AI 有关系吗？** → 直接相关最好；间接相关需要找到连接点；完全无关 → 跳过
2. **我有真实经历吗？** → 有最好。没有就从老韩的日常找对应——楼下邻居、朋友公司、花过的钱、踩过的坑。基于老韩人设（产品老兵、token大户、十几年互联网）构造合理感受
3. **路人为什么要在意？** → 观众关心的是：能不能赚钱、会不会失业、省不省钱、好不好用。把技术话题翻译成生活语言

## 角度库

| 角度 | 适用场景 | 老韩的优势 |
|------|---------|-----------|
| 就业焦虑 | AI取代岗位、裁员、职业转型 | 你是用AI替代人工的人 |
| 消费/省钱 | AI工具价格、token费用、性价比 | 月烧大量token的真实用户 |
| 教育/成长 | AI学习、能力提升、认知升级 | 从零自学到搭建系统的亲历者 |
| 职业选择 | 行业趋势、跳槽、创业 | 十几年互联网+AI产品视角 |
| 日常场景 | AI在生活中的应用 | 楼下邻居、朋友公司、日常消费 |
| 反常识打假 | AI谣言、过度宣传、虚假承诺 | 花过真金白银试过工具的人 |

**角度选择优先级：反常识性 > 情绪共鸣 > 场景包装潜力。**

## 两句话标题公式（＜30字）

标题必须是两句话，用逗号、句号或破折号连接：

**第一句（主题句，15-20字）：** 说清楚发生了什么 + 你的立场。

**第二句（情绪句，10-20字）：** 用生活化语言拉情绪、反转、或制造悬念。必须有具体画面/数字/场景。

**好的例子：**
- "35岁被AI取代通知，拒绝降薪被开除，法院判了——但我想说句真话"
- "孩子抱AI聊3小时，真不是AI的错，也许他问人活着有什么意思，你不该吓一跳"

**坏的例子：**
- "也许AI能力等级该是划分人群的时候了" — 抽象概念
- "范式转移""赛道消失" — 念出来不像跟朋友说话

**红线：** 总长＜30字，第二句必须存在，禁止抽象概念，标题念出来必须顺嘴。

## 写稿态度铁律（style.md 未覆盖）

style.md 管语感和技法，这里管态度、立场、情绪：

1. **前15秒必须亮出态度**：不只是信息冲突，要有立场
2. **老韩必须下场表态**：至少1处老韩自己的判断/骂人/质问
3. **身份反差是最大炸弹**：老韩立场跟大众预期冲突时，放大不要缩着说
4. **细节要具体到让人愤怒**：不能只说"他公司也这样"，要"设计部五个人裁到两个，活没少，工资降15%"
5. **结尾不能说教**：换成有攻击性的收刀

## 爆款自检清单

**标题：**
- [ ] 两句话？第一句主题+冲突，第二句态度+情绪
- [ ] 第二句用生活语言？有具体画面/数字/场景？
- [ ] 总长＜30字？念出来顺嘴？

**内容：**
- [ ] 前15秒有态度吗？
- [ ] 有没有纯叙述无情绪的段落？
- [ ] 老韩亲自骂人/质问/下判断？
- [ ] 身份反差可以放大？
- [ ] 朋友/身边人故事够具体？
- [ ] 结尾是"说到心坎里"而非"有道理"？
- [ ] 立场没摇摆？→ 选一边站死
- [ ] 有促评论的互动钩子？

## 标题破圈规则

- **禁止在标题里放技术词**：claude code、openclaw、skill、token、agent 等（教程类除外）
- **标题公式优先级**：反常识+悬念 > 场景+渴望 > 数字+悬念 > 冲突+反转
- **场景型标题是最强破圈武器**："饺子馆玩AI秒变skill"（3916播放，超均值80%）
- **标题自检：** 给不看 tech 推送的人看这个标题，他能理解在说什么吗？不能 → 改标题

**反面教材：**
- "GLM5.1降智真相"（440播放）→ 圈内标题，路人无感
- "claude code配置教程"（1771播放）→ 技术术语天花板

## 竞品缺口分析

选题时搜索同赛道博主，找到没人覆盖的角度：
- 5个博主都在讲同一热点 → 拥挤，找更反常识的角度或放弃
- 热点没有AI博主在讲但大众关注 → **蓝海选题**，优先做

## 发布策略

- **发布时间**：晚高峰 19:00-20:00（均播 3208，是其他时段 1.8 倍）
- **标题类型**：场景型播放爆发力最强，数字型最稳定
- **时长**：完播率 > 时长。精华优先于字数

## 前3秒法则

- 前3秒必须有冲突：直接甩出负面痛点、反常现象、或认知冲突
- 固定开场白"你有没有这种感觉，"之后立刻接冲突
- 标题关键词在前3秒内必须出现
- 完播率 > 时长：能用60秒说清的不要拖到120秒
